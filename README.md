# ComicVerse - HQ E-commerce  

**ComicVerse** is a full-featured e-commerce platform designed for buying and selling comic books. This project provides a robust backend, a dynamic frontend, and a secure database to handle transactions, user accounts, and inventory management.  

### Features:  
- 🛒 **Shopping Cart & Checkout** – Seamless purchasing experience with multiple payment options.  
- 📚 **Catalog Management** – Browse, search, and filter through a vast collection of comics.  
- 🔐 **User Authentication** – Secure login, registration, and profile management.  
- 📦 **Order Tracking** – Follow the status of your purchases in real-time.  
- ⭐ **Reviews & Ratings** – Users can review and rate comics.
